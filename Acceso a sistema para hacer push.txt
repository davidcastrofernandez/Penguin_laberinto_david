comando para pushear documentos a githup
git add .
git commit -m "el mensaje"
git push     alzar a la plataforma (para empujar el codigo del local al remoto)
git pull     bajar de la plataforma(para estirar el codigo del remoto al local)

comandos para navegar terminal
ls o dir  (para ver los archivos existentes)
cd nombre_de_archivo  (para entrar en el archivo)
cd . .  (para salir del archivo)
